# Braccio Robotico 3D

## Descrizione
Progetto di tirocinio per la realizzazione di un braccio robotico stampato in 3D,
controllato tramite Arduino e servomotori, con interfaccia software in Python.

## Obiettivi
- Progettare e stampare in 3D i componenti del braccio
- Programmare i servomotori tramite Arduino
- Sviluppare un software di controllo in Python

## Strumenti utilizzati
- Visual Studio Code
- Arduino IDE
- Fusion 360
- Bambu Studio
- Python 3.12
- GitHub Desktop
